PERSONALIZAÇÃO DE CARACTERÍSTICAS

Idioma - Altere o aplicativo para seu idioma nativo, com suporte para 18 idiomas.
Avatar - Carregue sua foto para aparecer como avatar no quadro.
Notificação - Assista a opção de uma placa, lista ou cartão para obter notificação sobre as mudanças de evento.
Arquivo Board - Opção para arquivar uma placa quando você concluir um projeto. Opção também está disponível para recuperar as placas arquivadas.
Etiquetas e Pesquisa - Você pode filtrar sua pesquisa em uma placa com base em um rótulo ou membro. Isso ajuda a listar as tarefas que podem precisar de atenção.
Multi-Selection - Opção para fazer atividade em massa como mover cartões, etc.
Mude a cor do fundo da placa - mude do defeito “que sente azul” a um toque personalizado.
Email Invites - Convide membros da equipe para a sua placa usando seu email

CARACTERÍSTICAS ADICIONAIS

Membros da Diretoria - Adicionar membros da placa aos cartões e atribuir-lhes tarefas ou manter em loop.

Etiquetas - Adicionar etiquetas para filtragem e visualização de dados

Anexos - Adicione anexos ao cartão para fornecer informações adicionais às tarefas.

Sistema Kanban - Características Básicas

Conselhos - Um Conselho representa um projeto. Se você está desenvolvendo um produto ou gerenciando problemas no suporte de produção, o sistema kanban board é o lugar para organizar suas tarefas.
Listas - As listas organizam os cartões e são usadas para criar um fluxo de trabalho.
Cartões - Esta é uma unidade fundamental e representa uma tarefa ou história de usuário.

Com Sistema kanban você pode adicionar várias placas, listas e cartões.

Princípios básicos do Kanban

3 princípios essenciais

Visualize o trabalho (workflow): Ver todos os itens em contexto uns dos outros pode ser muito informativo

Limitar a quantidade de trabalho em andamento (WIP): Ajuda a equilibrar a abordagem baseada em fluxo para que as equipes não comecem e se comprometam a trabalhar demais de uma vez

Melhorar o fluxo: Quando algo é terminado, a próxima coisa mais alta do backlog é puxado para o jogo
